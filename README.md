# MRI-brain-tumor-detector-and-Heatmap-using-Explainable-AI-
An AI-powered MRI brain tumor detector using ResNet50 transfer learning and Grad-CAM explainable AI. Classifies MRI scans into Glioma, Meningioma, No Tumor, and Pituitary with a Streamlit web app.


👉 https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
